## 📋 概要

TravelNest Demo は、ホテル予約サービスのフロントエンドデモアプリケーションです。

## 🚀 起動方法

### ローカルサーバーを使う場合（推奨）

```bash
# Python 3
python -m http.server 8080

# Node.js (npx)
npx serve .

# VS Code: Live Server 拡張機能を使う場合は index.html を右クリック → Open with Live Server
```

ブラウザで `http://localhost:8080` を開いてください。

### ファイルを直接開く場合

`index.html` をブラウザで直接開くことも可能ですが、一部機能（LocalStorage等）は `file://` プロトコルでは制限される場合があります。

    └── master.js       # マスターデータ（ユーザー・ホテル・予約）

## ✨ 主要機能

| 機能 | 説明 |
|------|------|
| 🔍 ホテル検索 | 目的地・日程・人数・キーワードで検索（ゲスト利用可） |
| ♾️ 無限スクロール | 検索結果を5件ずつ非同期ロード（LoadingアイコンUI） |
| 🏨 ホテル詳細 | 別ウィンドウ表示・先2週間の空き状況グリッド |
| 📅 日程選択/空き確認 | 日程未指定で予約ボタン押下時の中間画面 |
| 🛒 一覧から直接予約 | 検索結果の各ホテルカードから直接予約可 |
| ✅ 予約フロー | 入力 → 確認 → 完了の3ステップ / 予約番号発行 |
| ❤️ お気に入り | ホテルの保存・解除・一覧表示 |
| 📋 予約履歴 | ステータスフィルター・キャンセル（前日まで） |
| 🔐 認証 | ログイン・セッション管理・未ログイン時のモーダル誘導 |
| 📱 レスポンシブ | モバイル対応デザイン |
| 💾 データ永続化 | LocalStorage によるブラウザ内保存 |

---

## ⚙️ ビジネスルール

1. **検索はゲスト（未ログイン）でも利用可能**
2. **予約・お気に入りはログイン必須**（未認証時はログインモーダルを表示）
3. **チェックインは予約日の翌日以降のみ**
4. **同一ユーザーの期間重複予約は不可**（確認画面移行前にチェック）
5. **キャンセルはチェックイン前日23:59まで**（当日以降は取消不可）
