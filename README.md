## 🚀 起動方法

### ローカルサーバーを使う場合（推奨）

```bash
# Python 3
python -m http.server 8080

# Node.js (npx)
npx serve .

# VS Code: Live Server 拡張機能を使う場合は index.html を右クリック → Open with Live Server
```

ブラウザで `http://localhost:8080` を開いてください。

### ファイルを直接開く場合

`index.html` をブラウザで直接開くことも可能ですが、一部機能（LocalStorage等）は `file://` プロトコルでは制限される場合があります。

```

├── login.html          # ログイン
├── register.html       # 会員登録（デモ）
├── mypage.html         # マイページ（予約履歴・お気に入り）
├── help.html           # ヘルプ・FAQ・テストアカウント一覧
├── css/
│   └── style.css       # グローバルスタイル
├── js/
│   ├── app.js          # アプリコア（状態管理・認証・予約・API）
│   └── layout.js       # ヘッダー・フッター・免責バナー生成
└── data/
    └── master.js       # マスターデータ（ユーザー・ホテル・予約）
```

## ✨ 主要機能

| 機能 | 説明 |
|------|------|
| 🔍 ホテル検索 | 目的地・日程・人数・キーワードで検索（ゲスト利用可） |
| ♾️ 無限スクロール | 検索結果を5件ずつ非同期ロード（LoadingアイコンUI） |
| 🏨 ホテル詳細 | 別ウィンドウ表示・先2週間の空き状況グリッド（iframe） |
| 📅 日程・空き確認 | 左カラム：日程/人数入力、右カラム：部屋タイプ別空き結果の2カラムレイアウト |
| 📆 日付入力 | テキスト入力（yyyy/mm/dd・yyyy-mm-dd）とカレンダーピッカー（📅ボタン）の両方に対応 |
| 🛒 予約フロー | 日程・空き確認 → 予約入力 → 内容確認 → 完了（支払い入力なし） |
| ❤️ お気に入り | ホテルの保存・解除・一覧表示 |
| 📋 予約履歴 | ステータスフィルター・ランダム表示・キャンセル（前日まで） |
| 🔐 認証 | ログイン・即時会員登録・セッション管理・未ログイン時のモーダル誘導 |
| ⏳ Loading演出 | ログイン・検索・空き確認で3〜5秒のオーバーレイ表示（操作ロック付き） |
| 📱 レスポンシブ | モバイル対応デザイン（日程選択画面はモバイルで縦積み） |
| 💾 データ永続化 | LocalStorage によるブラウザ内保存（ウィンドウ間でセッション共有） |

## ⚙️ ビジネスルール

1. **検索はゲスト（未ログイン）でも利用可能**
2. **予約・お気に入りはログイン必須**（未認証時はログインモーダルを表示）
3. **チェックインは予約日の翌日以降のみ**
4. **同一ユーザーの期間重複予約は不可**（予約確定時にチェック）
5. **キャンセルはチェックイン前日23:59まで**（当日以降は取消不可）
6. **予約フローに支払い入力なし**（デモのため不要）
7. **ホームページの日付入力は初期未設定**（日程はdate-select画面で選択）
